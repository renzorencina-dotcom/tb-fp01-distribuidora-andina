# Distribuidora Andina Fresh S.A.C.

Proyecto en Python para automatizar el registro de pedidos y la validación de stock.

## Objetivo
Desarrollar un sistema que permita registrar productos, clientes y pedidos, procesar automáticamente la información y generar reportes básicos.

## Tecnologías
- Python
- PyCharm
- Git
- GitHub
- Tkinter (opcional)

## Estructura general
- `models/`: entidades del sistema
- `services/`: lógica del negocio
- `ui/`: interfaz de usuario
- `utils/`: funciones auxiliares
- `data/`: archivos CSV
- `docs/`: documentación técnica
